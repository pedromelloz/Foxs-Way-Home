# Foxs-Way-Home
Projeto Final de Desenvolvimento de Jogos Digitais - Jogo de plataforma 2D sobre uma raposa que tenta voltar para casa.

## Intgrante
Pedro Miguel Kojo de Melo

## Descrição
Fox's Way Home é um jogo 2D em que uma pequena raposa presica superar seus obstáculos e desafios no parkour para encontrar o caminho de volta para casa.

## Gênero
Plataforma 2D e Aventura.

## Público-Alvo
Adolescentes e jovens que gostam de jogos de plataforma e exploração.

## Tecnologias utilizadas
- Unity 6
- C#
- Visual Studio Code
- GitHub
- Trello

## Como executar
1. Abra a página do jogo no Itch.io.
2. Clique em .
3. Espere alguns segundos para o jogo carregar no navegador.
4. Depois do carregamento, clique na tela do jogo para começar.
